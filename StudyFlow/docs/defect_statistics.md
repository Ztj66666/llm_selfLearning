## 3. 缺陷分析
- **主要问题来源**: 前后端接口定义不一致（路径前缀、端口号）。
- **改进措施**: 建立了统一的 API 路由规范 (`/api/*`) 和动态配置管理 (`api.ts`, `.env`)，有效减少了此类问题。
